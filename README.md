# Law Enforcement Grant Finder

An independent open-source dashboard for locating publicly available grant opportunities from approved government sources.

## Features
- Search and filter grant opportunities
- Official source links
- Eligibility and deadline summaries
- Daily approved-source availability checks
- Historical source-health records
- Automated debugger and unit tests
- GitHub Pages ready

## Important notice
This project does not guarantee eligibility, funding, deadlines, or award decisions. Verify all details on the official source before applying. Do not upload sensitive law-enforcement information, credentials, investigative data, or personally identifiable information.

## Run locally
```bash
python -m http.server 8000
```
Then open `http://localhost:8000`.

## Automation
The daily updater checks approved source availability and records timestamps. It does not scrape restricted systems or submit applications.
